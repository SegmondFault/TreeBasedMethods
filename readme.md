# TreeBasedMethods

This repository contains my Python work on tree-based machine learning methods, with a focus on implementing and testing:

- Decision Trees
- Random Forest
- AdaBoost

## Important setup note

The dataset file:

`PhiUSIIL_Phishing_URL_Dataset.csv`

must be placed in the main repository directory before running the notebook.

Example structure:

TreeBasedMethods/
- PhiUSIIL_Phishing_URL_Dataset.csv
- RandForest_Adaboost.ipynb
- README.md

## Files

- `RandForest_Adaboost.ipynb`  
  Main notebook containing the implementation, experiments, and evaluation.

- `RandForest_Adaboost_Live_WIP.ipynb`  
  Development notebook used during the working process.

## Project overview

The aim of this project is to explore how tree-based machine learning methods work in practice by building and testing them in Python.
