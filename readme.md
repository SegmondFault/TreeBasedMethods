# TreeBasedMethods

This project demos a pure Python implementation of various tree based ML algorithims:
-Decision Trees
-Random Forest
-AdaBoost

This is a reference implementation, not a production grade tool.

## Important setup note

The dataset file:

`PhiUSIIL_Phishing_URL_Dataset.csv` (which can be sourced from Kaggle)

must be placed in the main repository directory before running the notebook.

Example structure:

TreeBasedMethods/
- PhiUSIIL_Phishing_URL_Dataset.csv
- RandForest_Adaboost.ipynb
- README.md
